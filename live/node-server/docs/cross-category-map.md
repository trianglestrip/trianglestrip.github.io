# 全平台分类跨平台映射

记录「统一分类名 → 各平台 cid」的映射关系，供全平台首页、分类页、侧栏推荐等使用。

## 相关文件

| 文件 | 说明 |
|------|------|
| `scripts/sync-category-cross-map.mjs` | 同步脚本；`MANUAL` / `GROUP_MANUAL` 为手工维护条目 |
| `src/browse/cross-categories.data.ts` | 完整映射数据（自动生成，勿手改） |
| `src/browse/hot-cross-categories.ts` | 全平台「热门」展示顺序 |
| `src/browse/cross-key-aliases.ts` | 旧冗长 key → 简短 key 别名 |
| `web/src/utils/categoryDisplay.js` | 前端 fallback，与 data.ts 同步 |
| `web/src/config/hotCrossCategories.js` | 前端热门 key 列表（与 hot-cross-categories.ts 同步） |
| `web/src/utils/crossCategoryItems.js` | `/all/category` 分类网格数据（含 API 兜底） |

更新流程：

```bash
cd live/node-server
npm run sync:cross-map
```

修改映射时：先改 `sync-category-cross-map.mjs` 里的 `MANUAL` / `HOT_CROSS_CATEGORY_KEYS` / `cross-key-aliases.ts`，再执行同步。

## Key 命名约定

- **热门分类**：简短可读 key（拼音缩写或英文名），如 `sjz`、`yanzhi`、`lol`
- **非热门分类**：同步脚本按各平台 cid 自动生成 `g{douyu}_{huya}_{douyin}_{bilibili}` 形式（仅内部使用）
- **路由**：`/all/category/{key}`，例如 `/all/category/sjz`
- **API**：`GET /api/cross-rooms?key={key}`

全平台分类图标取自虎牙：`https://huyaimg.msstatic.com/cdnimage/game/{huyaCid}-MS.jpg`

## 新增 / 修改映射 checklist

1. 在 `sync-category-cross-map.mjs` 的 `MANUAL` 增加或修改条目（指定简短 `key`）
2. 若需出现在全平台热门，更新 `hot-cross-categories.ts` 的 `HOT_CROSS_CATEGORY_KEYS`
3. 若替换已有 key，在 `cross-key-aliases.ts` 添加旧 → 新别名
4. 运行 `npm run sync:cross-map`
5. 运行 `npm run verify:cross-map` 做完整性校验
6. 更新本文档对应表格
